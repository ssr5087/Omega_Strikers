# 역할
너는 언리얼 엔진 5.7버전 게임 개발자야

## 목표
오메가 스트라이커스 모작게임을 만들고 있어
골을 넣으면 UI 연출 나오고 각자의 TeamPlayerStart 지점으로 돌아가야해

## 현재상황
1. 처음 스폰될때 Team과 상관없이 랜덤으로 TeamPlayerStart 지점에러 랜덤 스폰 되고 있어
2. 이렇게 되면 같은 팀끼리 같은 장소에서 시작해야하는데
3. 한장소에 다른 팀과 섞여있는 상황이야

## 해야할 일
1. 왼쪽이 블루팀, 오른쪽이 레드팀이야
2. 골대도 왼쪽이 블루팀 골대, 오른쪽이 레드팀 골대야
3. 레드팀과 블루팀의 스폰위치를 나눠서 적용하고싶어
4. 골을 넣고 UI가 나오면 다시 TeamPlayerStart 지점에서 시작하고싶어
5. 같은팀이면 캐릭터가 겹쳐있어도 좋으니 신경 안써도 돼
6. 다하고 한글로 코드리뷰 해줘

### 용어
Core가 골이야 (Core = 골)

## 봐야할 파일
OSGameMode에서 작업하고 있는데
OSCharSelectGameMode에서 팀과 캐릭터가 정해지고 넘어오는거같아
다른 파일도 봐도 좋아
