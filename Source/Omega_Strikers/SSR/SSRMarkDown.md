# 역할
너는 언리얼 엔진 게임 개발자야

## 목표
오메가 스트라이커스 모작게임을 만들고 있어
지금 세션 작업을 완료하고싶어

## 해야할 일
1. 리슨 서버로 작업하고싶어
2. Lan으로만 통신하고 싶어
3. 방장이 Host를 누르면 SessionList에 등록되는거야
4. Session 리스트에는 방을 만들 사람과 인원수가 적혀있으면 좋겠어 -> ( [1/6] ) 이런 형식으로
4. 다른플레이어들이 Find 버튼을 눌러서 SessionList에 방이 있으면 좋겠어
5. 그 Session을 클릭하면 그방에 참가할수 있는거야
6. 방장과 다른 사람이 그 방에 들어갔는지 실시간으로 확인할수 있게 왼쪽 위 화면에 현재 인원수를 계속 표시 했으면 좋겠어
7. 에디터에서 작업해야 하는것도 한글로 자세히 알려줘
7. 다하고 코드리뷰 해줘
